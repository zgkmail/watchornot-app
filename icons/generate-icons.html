CineSense App Icons

How to Generate Icons:

  1. Take screenshots of each icon size below (or use a screenshot tool to capture exact dimensions)
  2. Save them with the filenames shown below each icon
  3. Place all icons in the /icons directory
  4. Alternatively, use an online tool like RealFaviconGenerator

Or use this command to generate basic icons (requires ImageMagick):

# Install ImageMagick first (if not installed)
# macOS: brew install imagemagick
# Ubuntu: sudo apt-get install imagemagick

# Generate icons from a source image (512x512 recommended)
for size in 72 96 120 128 144 152 180 192 384 512; do
  convert source-icon.png -resize ${size}x${size} icon-${size}.png
done
